## 📋 Sobre o Projeto
Aplicativo mobile desenvolvido em **React Native com Expo**, contendo um formulário de cadastro simples.  
O usuário preenche os campos e visualiza os dados inseridos na própria tela após o envio.

## 🚀 Funcionalidades
- Formulário com campos de Nome, Curso, Disciplina e Descrição
- Exibição dos dados preenchidos após clicar em Enviar
- Controle de estado com `useState`
- Inicialização registrada com `useEffect`

## 🛠️ Tecnologias Utilizadas
- React Native
- Expo SDK 54
- react-native-safe-area-context

## 🗂️ Estrutura do Projeto
```
CP1-MOBILE/
├── assets/         — Imagens e ícones
├── App.js          — Componente principal
├── app.json        — Configurações do Expo
└── index.js        — Entry point
```

## ▶️ Como Rodar o Projeto
```bash
npm install
npm start
```
Escaneie o QR code com o aplicativo **Expo Go** no celular ou rode no emulador Android.
